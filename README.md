# Commit Convention

Commit Message 규칙 (Commit Convention)
`feat` : 새로운 기능 추가
`fix` : 버그 수정
`docs` : 문서 관련(주석 및 README 수정)
`style` : 스타일 변경 (포매팅 수정, 들여쓰기 추가, …)
`build` : 빌드 관련 파일 수정
`ETC`: 기타 수정

```txt
(유형), 커밋 제목

(내용을 입력합니다. 3줄이하 생략가능합니다.)
```
